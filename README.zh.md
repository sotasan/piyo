## 功能特性

- ⚡ GPU 加速渲染
- 🍎 原生 macOS 外观与体验
- ✨ Claude Code 和 Codex CLI 集成
- 🐚 支持 Bash、Zsh、Fish 和 Nushell
- 🎨 Shiki 驱动的主题
- ⚙️ 基于 TOML 的用户配置

## 安装

### Homebrew

```sh
brew install --cask sotasan/tap/piyo
```

### 手动安装

从[发布页面](https://github.com/sotasan/piyo/releases/latest)下载最新版本。
