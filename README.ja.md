## 特長

- ⚡ GPU アクセラレーションによる描画
- 🍎 ネイティブな macOS の見た目と操作感
- ✨ Claude Code 連携
- 🐚 Bash、Zsh、Fish、Nushell に対応
- 🎨 Shiki ベースのテーマ
- ⚙️ TOML による設定

## インストール

### Homebrew

```sh
brew install --cask sotasan/tap/piyo
```

### 手動

[リリースページ](https://github.com/sotasan/piyo/releases/latest)から最新版をダウンロードしてください。
